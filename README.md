# puffer_discord
This project is a Discord bot for managing PufferPanel game server instances.

Use Bun

```bash
bun install
```

To run:

```bash
bun run index.ts
```

Use Docker:

```bash
docker compose up -d
```

This project was created using `bun init` in bun v1.1.40. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
